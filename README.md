The code repository is for the paper [Lookahead Counterfactual Fairness](https://openreview.net/forum?id=kmHq3pKIlj&referrer=%5BAuthor%20Console%5D%2Fgroup%3Fid%3DTMLR%2FAuthors%23your-submissions).

```law_experiment``` contains the code for the real world dataset experiments.

```Theorem_5_1``` to ```Theorem_5_4``` contains the synthetic experiments for the four theorems.
